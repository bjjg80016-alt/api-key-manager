🌍 环境变量

您也可以使用环境变量来设置API密钥：

export OPENAI_API_KEY=sk-your-openai-key export ANTHROPIC_API_KEY=sk-ant-your-anthropic-key export NEWS_API_KEY=your-news-api-key

🚀 部署选项

1. 本地部署

# 启动命令行工具 python src/api_key_manager.py # 启动Web界面 python src/web_interface.py # 启动主程序 python start.py

2. Docker部署

# 构建镜像 docker build -t api-key-manager . # 运行容器 docker run -d -p 8080:8080 api-key-manager # 使用Docker Compose docker-compose up -d

3. 访问地址

Web界面: http://localhost:8080 API文档: http://localhost:8080/docs 健康检查: http://localhost:8080/health

📋 系统要求
